# Notes

## How do you decide which unit to use?

### Font sizes
- Use `rem`
- No compounding
- More predictable

### Margin & padding
- Use `em`
- Proporional scaling
- Based on element's current font-size

### Line height
- Unitless number values

### Width
- Use `%`
- Flexible containers
- Flexible images

### Max-width
- Use `px`
- Easier to manage
- Ensures consistency
